# 🚚 Delivery Profit API (Django Backend)

Backend da aplicação **Delivery Profit**, desenvolvido com Django + Django REST Framework, focado em autenticação via cookies HttpOnly e gerenciamento de operações de logística.

---

## 🚀 Tecnologias

- Python 3.13
- Django 6+
- Django REST Framework
- PostgreSQL
- django-cors-headers
- Auth via Token + HttpOnly Cookies

---

## 📦 Funcionalidades

- 🔐 Autenticação com login e logout
- 🍪 Cookies HttpOnly para segurança
- 👤 Registro de usuários
- 👤 Endpoint de usuário autenticado (`/me`)
- 🚛 Estrutura base para rotas e veículos
- 🌐 API REST estruturada

---

## 🧠 Arquitetura

Frontend (React + Vite)
↓
API (Django REST)
↓
PostgreSQL (Neon / local)

---
